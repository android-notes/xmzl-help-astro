    如果您需要使用Edge浏览器进行网页自动化需要先安装Edge浏览器及小秘助理Edge插件。
    >- CRX插件文件下载 <A href='https://static.xmzl.work/downloads/chrome-plugin.crx'></A>
    1. 使用Edge浏览器打开 <A href="edge://extensions/"></A>
    2. 打开`开发者模式`
    <Image src="/edge-extensions.png"/>
    3. 拖动`chrome-plugin.crx`文件到Edge中
    <Image src="/add-edge-extensions.png"/>
    4. 点击弹窗中的`添加扩展程序`
    <Image src="/add-edge-extensions-dialog.png"/>
    5. 安装成功后会显示在Edge中，请确保处于`开启`状态
    <Image src="/add-edge-extensions-finish.png"/>
    6. `重启Edge浏览器`就可以正常使用网页自动化了
</DocsLayout>
