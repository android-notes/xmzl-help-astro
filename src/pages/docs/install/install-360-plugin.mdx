    如果您需要使用360极速浏览器X进行网页自动化需要先安装360极速浏览器X及小秘助理插件。
    >- CRX插件文件下载 <A href='https://static.xmzl.work/downloads/chrome-plugin.crx'></A>
    1. 使用360极速浏览器X打开 <A href="chrome://extensions/"></A>
    2. 打开`开发者模式`
    3. 拖动`chrome-plugin.crx`文件到360极速浏览器X中
    4. 点击弹窗中的`添加扩展程序`
    5. 安装成功后会显示在360极速浏览器X中，请确保处于`开启`状态
    6. `重启360极速浏览器X`就可以正常使用网页自动化了
